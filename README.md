# VoiceToText

A native macOS dictation app. Hold a global hotkey, speak, release, and the transcribed text is typed into whatever app is focused. Local-first, runs offline on the Apple Neural Engine.

## Download

[**Download VoiceToText App**](https://github.com/gug007/voice-to-text/releases/latest) — grab the `.dmg` from the assets list.


## Install

1. Open the DMG and drag **VoiceToText** to `/Applications`.
2. Launch it
3. Grant Microphone and Accessibility permissions when prompted.
4. Hold the hotkey, speak, release.
